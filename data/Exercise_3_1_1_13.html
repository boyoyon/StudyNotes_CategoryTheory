Category Theory for the Sciences
科学のための圏論

Exercise 3.1.1.13. 演習 3.1.1.13.

For every set \(A\) there is some relationship between the following three sets:
\(HomSet(A,X),\quad HomSet(A,Y)\) and \(HomSet(A,X×Y)\).
What is it?
任意の集合 \(A\) に対して、以下の3つの集合の間には何らかの関係が存在します。
\(HomSet(A,X),\quad HomSet(A,Y)\) および \(HomSet(A,X×Y)\)。
それはどのような関係でしょうか？

Hint: This problem is somewhat recursive in that you will use products in your formula.
ヒント：この問題は、数式の中に積の計算が含まれるという点で、ある種の再帰的な性質を持っています。

●解答
There is an isomorphism
同型性がある \[ Hom_{Set}(A,X×Y)\overset{\simeq}{\longrightarrow}Hom_{Set}(A,X)×Home_{Set}(A,Y) \] In an attempt to make this more concrete, suppose we have a height function from the set \(P\) of people to the set Z of integers and a name function from the set of people to the set \(S\) of strings. That is, we have an element of \(Hom_{Set}(P, \mathbb{Z})\) and an element of \(Hom_{Set}(P, S)\). From this we get an element of \(Hom_{Set}(P, \mathbb{Z})×Hom_{Set}(P, S)\). That is, the two pieces of information combine into a single piece of information if we pack the height and the name into a single datum, i.e., an element of \(\mathbb{Z}×S\).

これをより具体的に説明するために、人々の集合 \(P\) から整数の集合 \(\mathbb{Z}\) への身長関数と、人々の集合から文字列の集合 \(S\) への名前関数があると仮定しましょう。つまり、\(Hom_{Set}(P, \mathbb{Z})\) の要素と \(Hom_{Set}(P, S)\) の要素があるということです。これから、\(Hom_{Set}(P, \mathbb{Z})×Hom_{Set}(P, S)\) の要素が得られます。つまり、身長と名前という2つの情報を1つのデータ、すなわち \(\mathbb{Z}×S\) の要素としてまとめれば、これらの情報は1つの情報に統合されるということです。
