Exercise 3.1.2.15. 演習 3.1.2.15.

Following Section 3.1.1.16, devise a naming system for coproducts, the inclusions, and the universal maps. Try it out by making an olog (involving coproducts) that discusses the idea that both a .wav file and an .mp3 file can be played on a modern computer. Be careful that your arrows are valid (see Section 2.3.2.1).
セクション3.1.1.16に続いて、コプロダクト、包含写像、および普遍写像のための命名規則を考案してください。現代のコンピュータでは.wavファイルと.mp3ファイルの両方を再生できるという考え方を説明するオログ（コプロダクトを含む）を作成して、その命名規則を試してみてください。矢印が有効であることを確認してください（セクション2.3.2.1を参照）。

●解答
 Given two objects \(c, d\) in an olog, there is a canonical label “\(c \sqcup d\)” for their coproduct \(c \sqcup d\), written in terms of the labels “\(c\)” and “\(d\).” Namely,
olog 内の2つのオブジェクト \(c, d\) が与えられたとき、それらの余積 \(c \sqcup d\) には標準的なラベル “\(c \sqcup d\)” があり、これはラベル “\(c\)” と “\(d\)” を使って表されます。つまり、 \[ \langle\langle c \sqcup d \rangle\rangle := \text{"}\langle\langle c \rangle\rangle \text{("}\langle\langle c \rangle\rangle\text{"と表記される) or }\langle\langle d \rangle\rangle\text{("}\langle\langle d \rangle\rangle\text{"と表記される)"} \] The inclusions \(c → c \sqcup d ← d\) can be labeled “after being tagged “\(c\)” is” and “after being tagged “\(d\)” is” respectively.
包含関係 \(c → c \sqcup d ← d\) は、それぞれ「「\(c\)」とタグ付けされた後は」および「「\(d\)」とタグ付けされた後は」とラベル付けすることができる。
