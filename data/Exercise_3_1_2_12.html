Exercise 3.1.2.12. 演習 3.1.2.12.

Write the universal property for coproduct, in terms of a relationship between the following three sets:
余積に関する普遍性について、以下の3つの集合間の関係を用いて記述しなさい。 \[ Hom_{Set}(X,A) \quad Hom_{Set}(Y,A) \quad Hom_{Set}(X \sqcup Y,A) \]

●解答
 \[ Hom_{Set}(X \sqcup Y,A)→\simeq Hom_{Set}(X,A)×Hom_{Set}(Y,A) \] To assign an \(A\) value to each element of \(X \sqcup Y\), you can delegate responsibility: have one person assign an \(A\) value to each element of \(X\), and have another person assign an \(A\) value to each element of \(Y\). One function is equivalent to two.
\(X \sqcup Y\) の各要素に \(A\) 値を割り当てるには、責任を分担することができます。つまり、一人が \(X\) の各要素に \(A\) 値を割り当て、別の人が \(Y\) の各要素に \(A\) 値を割り当てるようにすればよいのです。一つの機能は二つの機能に相当します。
